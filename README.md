# DeafLink 

**DeafLink** is a lightweight, highly-optimized accessibility mod for *R.E.P.O.* designed to help deaf and hard-of-hearing players. 

It seamlessly translates the game's 3D spatial audio into a visual radar system in real-time allowing you to "see" important audio cues perfectly hugging the edges of your screen. 
**Please note that it currently only works in Singleplayer and for the host in Multiplayer.**

## Features:
* **True Radar Projection:** Sound origins are accurately projected to the edges of your screen. If an explosion happens to your right, the icon pops up at the right edge of the screen.
* **Smart Proximity Tracking:** Automatically groups footsteps from the same monster into a single smoothly gliding icon. Tracks multiple enemies flawlessly with zero UI spam.
* **Categorized Icons:** Features 32 distinct, high-quality PNG icons covering every single monster type in the game, plus universal icons for Weapons, Explosions, and Item Breaks.
* **Dynamic Range:** Different range of sound visualization for different types of sound (i.e An explosion sound will be visualized from far away but the same won't apply to the humming of the huntsman)
* **Extremely Optimized:** Built entirely from the ground up to use memory pooling and zero Garbage Collection allocations. Your framerate will not drop.

## Installation:
1. Ensure you have **BepInEx 5** installed.
2. Download the latest `DeafLink.zip` and extract its contents into your `BepInEx\plugins` folder.
3. Launch the game!
