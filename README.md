# DeafLink 

**DeafLink** is a lightweight, highly-optimized accessibility mod for the game *R.E.P.O.* designed to help deaf and hard-of-hearing players. 

It seamlessly translates the game's 3D spatial audio into a visual radar system, allowing you to "see" important audio cues perfectly hugging the edges of your screen. 

## Features:
* **True Radar Projection:** Sound origins are mathematically projected to the edges of your screen.
* **Categorized Icons:** Features 32 distinct, high-quality PNG icons covering every single monster type in the game, plus universal icons for Weapons, Explosions, and Item Breaks.
* **Dynamic Range:** Different range of sound visualization for different types of sound (i.e An explosion sound will be visualized from far away but the same won't apply to the humming of the huntsman)
* **Extremely Optimized:** Built entirely from the ground up to use memory pooling and zero Garbage Collection allocations. Your framerate will not drop.

## Installation
1. Ensure you have **BepInEx 5** installed for R.E.P.O.
2. Download the latest `DeafLink.dll` from the Releases tab or Thunderstore.
3. Drop the `DeafLink.dll` file into your `BepInEx\plugins` folder.
4. Launch the game!
